Changelog

A history of changes to the awesome-wipeout library. For full details, visit the GitHub repository.

Release 5 2026-07-12

This release completes the Teams brand-guidelines matrix with the 2048 era and tidies how shared team emblems are shown.

New content

  • WipEout 2048 column — the matrix now runs through the 2012 prequel. Its five founding teams (AG Systems, Auricom, FEISAR, Piranha, Qirex) are marked as gaps: 2048 used its own distinct liveries, which aren't in the collection yet.
  • WipEout Pulse emblems — Pulse shares HD's team-select emblems, so its previously-empty cells are now filled in across the roster.

Improvement

  • Shared emblems shown once — where Pulse and HD carry the identical mark, it's rendered a single time in a centred, column-spanning cell instead of being repeated in both columns, with the logo sized so wide marks (e.g. EG.X) no longer balloon. Detail drawers and deep-links are preserved, with a combined "Pulse & HD" label.

Release 4 2026-07-12

This release expands the in-game reference gallery with two more teams.

New content

  • 49 new in-game reference screenshots — Tigron (23) and Van-Über (26), from the WipEout Omega Collection. Both teams now have their own sections in the gallery (headed by their HD-era emblems), bringing the in-game reference collection to 158 screenshots.

Release 3 2026-07-12

This release adds the Leagues page and makes the detail views across the site deep-linkable and cross-linked.

Features

  • Leagues page — a new sortable reference of every anti-gravity racing league: AGRC, F3600, F5000, F7200, F9000, FX300, FX350 and FX400. Each row shows the league's emblem, its game, real-world release year and in-game year, plus the marks and fonts that carry its identity; the table sorts by release or in-game year (which genuinely differ — WipEout HD is set before Pulse but shipped after it). Clicking a league opens a full-window lightbox with a details card, its marks and fonts, and the lore. Driven by the new data/leagues.toml.
  • Deep-linkable, navigable lightboxes — the Leagues, Marks, Fonts and Teams detail views now have shareable URLs (leagues.html#f9000, index.html#m/<cat>/<slug>, fonts.html#f/<slug>, teams.html#<team>--<series>), with working browser back/forward and, on the Leagues lightbox, left/right arrow navigation between leagues.
  • Cross-linking between pages — the Leagues lightbox links out to the full-screen Marks and Fonts pages for each asset (rather than duplicating downloads), and the Marks and Fonts lightboxes show "Featured in" back-links to every Team and League that uses that mark or font.

New content

  • Per-game metadata on each league — a Metacritic score and link, the platform list (e.g. PS Vita / PS4), and a link to the game's LaunchBox Games Database page.
  • Fonts identified per league — the typefaces behind each era's logos and cover art, including Amalgama, Bitsumishi, OCR-B, Orbit-B, 04b_19, DF So Gei W 9, SilverStream, Neo Sans and the fan recreations.

Improvement

  • "Source on GitHub" link added to the shared footer on every page.
  • Teams page brand cells are now deep-linkable — a URL per team × series that opens the drawer on load.

Release 2 2026-07-12

This release adds the Teams page, a large batch of WipEout HD team logos, and a set of site-wide improvements and fixes. Asset count grows from 200 to 223 marks.

Features

  • Teams page — a new brand-guidelines matrix (in the top navigation) laying out every racing team across the series eras, from WipEout through WipEout HD. Each cell shows the team's era logo, its identity colours (with official / sampled swatches), and SVG + PNG downloads. Driven by the new data/teams.toml.

New content

  • 12 colour WipEout HD team logos — recoloured HD / Fury team-select art: AG Systems, Assegai, Auricom, EG.X, FEISAR, Goteki 45, Harimau, Icaras, Mirage, Piranha, Qirex, Triakis.
  • 9 HD colour "lockups" — each team's colour device combined with its wordmark into a single mark: Assegai, FEISAR, Goteki 45, Harimau, Icaras, Mirage, Piranha, Qirex, Triakis.
  • New Icaras HD logo, plus a corrected FX300 league emblem and the FX350 "Fury" emblem as a reference-only stub.
  • Tigron and Van-Über added to the Teams matrix (Pure onwards).
  • Links — added the WipEout team history and lore video.

Improvement

  • Shared styles.css and analytics.js — CSS and analytics lifted out of per-page inline blocks into two shared files linked by every page, shrinking the generated HTML and keeping styling consistent.
  • Standardised title underline across pages.

Fixes

  • Title-logo transparency — series and game wordmarks now use SVG masks and no longer carry the opaque WipEout frame background, so they sit cleanly on any background.
  • Restored Auricom's missing red star elements in the Wip3out mark.
  • Re-exported the Piranha colour logo vector.

Release 1 2026-07-11

The initial public release of awesome-wipeout — the WipEout Vector Asset Library: a normalised, community-maintained collection of WipEout-universe logos, emblems, marks, fonts and reference imagery, published as a static site via GitHub Pages.

What's included

Marks — 200 vector assets (SVG + PNG) across 16 sections:

  • Game Titles — series/game wordmarks, WipEout (1995) → Omega Collection
  • Teams — per era: WipEout · WipEout 2097 · Wip3out · WipEout Pure · WipEout HD / 2048
  • WipEout HD — Wordmarks & Taglines
  • Pilots · Speed Classes · Game Modes · Weapons
  • Sponsors · Tracks & Venues · Leagues
  • Miscellaneous · Found — for review

Fonts — 34 referenced typefaces across 4 groups (Fan-made · Type foundries — free · Type foundries — commercial · System fonts). No font files are hosted — outlined specimen sheets only.

In-game reference — 109 screenshots (WipEout Omega Collection), grouped by team.

Links — a curated "awesome list" of WipEout art, builds, tools and communities.

About / Contributing / Licensing / Credits — project info and per-asset attribution.

Plus an aggregate tearsheet.pdf (multi-page, fully-vector tear sheet).
